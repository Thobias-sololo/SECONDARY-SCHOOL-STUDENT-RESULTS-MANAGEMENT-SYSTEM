# SECONDARY SCHOOL STUDENT RESULTS MANAGEMENT SYSTEM

PHP, MySQL, HTML, CSS, and JavaScript system for managing secondary school students, subjects, results, grades, rankings, report cards, class performance, profiles, and notifications.

## Technologies used

- CSS
- HTML
- Javascript
- PHP 
- MySQL 
- Web server such as Apache, XAMPP, 

## Main Roles

- Admin: manages users, students, teachers, subjects, assignments, results approval, reports, grades, and notifications.
- Teacher: enters marks for assigned classes and subjects, views performance, and sends notifications.
- Student: views own results, average, grade, position, report card, profile, and notifications.

## Main Files

- `index.php`: login page.
- `dashboard.php`: redirects users to the correct role dashboard.
- `admin/`: admin dashboard and management modules.
- `teacher/`: teacher dashboard, marks entry, performance, and notification pages.
- `student/`: student dashboard, results, and report card pages.
- `includes/results.php`: average, grade, and ranking logic.
- `database/schema.sql`: MySQL database structure and starter data.
